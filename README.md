# Juniper Mist Predictive Maintenance Demo

This is a simple **interactive HTML5 dashboard** simulating Juniper Mist AI predictive maintenance features.

## 💡 Features
- Simulated devices with auto-generated telemetry (CPU, temperature, errors, drop rates)
- Predictive risk scoring (OK / WARNING / CRITICAL)
- Alerts panel with live updates
- Device detail view and suggested actions
- Simulated "ticket creation" workflow

## 🚀 How to Use
1. Download the ZIP and extract it.
2. Open `index.html` in your web browser.
3. Click **Refresh** or enable **Auto Predict** to simulate live updates.
